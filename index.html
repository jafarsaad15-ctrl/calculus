📐 النهايات والاستمرارية

النهايات هي الأساس الذي يُبنى عليه التفاضل والتكامل. تساعدنا على فهم سلوك الدوال عند اقترابها من نقطة معينة.

تعريف

نهاية الدالة f(x) عند x تقترب من a تساوي L، أي lim(x→a) f(x) = L، إذا اقتربت قيم f(x) من L كلما اقتربت x من a من الجهتين.

الصيغة الأساسية
lim(x→a) f(x) = L
تُقرأ: نهاية f(x) عندما x تقترب من a تساوي L

📌 خصائص النهايات

الجمع lim(f+g) = lim f + lim g ↑ تُطبق على كل حد
الضرب lim(f·g) = lim f · lim g ↑ ضرب النهايتين
القسمة lim(f/g) = lim f / lim g ↑ بشرط lim g ≠ 0
الثابت lim(c·f) = c · lim f ↑ c ثابت حقيقي
✏️ مثال محلول

احسب: lim(x→2) (x² + 3x - 1)

  1. نعوّض مباشرة x = 2 لأن الدالة متصلة عند هذه النقطة
  2. (2)² + 3(2) - 1 = 4 + 6 - 1 = 9
  3. إذن النهاية = 9

⚠️ الحالات الخاصة

⚠️

عندما تكون النهاية من الشكل 0/0 أو ∞/∞، لا تعوّض مباشرة — استخدم التحليل، أو القسمة على أعلى أس، أو قاعدة لوبيتال.

✏️ مثال — شكل 0/0

احسب: lim(x→3) (x²-9)/(x-3)

  1. التعويض المباشر يعطي 0/0 — حالة غير محددة
  2. نحلل البسط: x²-9 = (x-3)(x+3)
  3. نبسّط: (x-3)(x+3)/(x-3) = x+3
  4. نعوّض: 3+3 = 6
  5. إذن النهاية = 6

🔗 الاستمرارية

تعريف

الدالة f متصلة عند النقطة a إذا تحققت ثلاثة شروط: (١) f(a) معرّفة، (٢) النهاية موجودة، (٣) lim(x→a) f(x) = f(a)

النهايات اللانهائية نهايات الدوال المثلثية lim(x→0) sinx/x = 1 قاعدة لوبيتال

📏 قواعد الاشتقاق

الاشتقاق يقيس معدل التغير الفوري للدالة — هو قلب التفاضل وأداة لا غنى عنها في الرياضيات والتطبيقات.

التعريف بالحد
f'(x) = lim(h→0) [f(x+h) - f(x)] / h
التعريف الأساسي للمشتقة من أول مبادئ

📋 جدول المشتقات الأساسية

الثابت d/dx (c) = 0 مثال: d/dx(5) = 0
الأس d/dx (xⁿ) = nxⁿ⁻¹ مثال: d/dx(x³) = 3x²
sinx d/dx (sinx) = cosx ↑ مثلثية أساسية
cosx d/dx (cosx) = -sinx ↑ لاحظ الإشارة السالبة
eˣ d/dx (eˣ) = eˣ ↑ تشتق من نفسها!
ln(x) d/dx (ln x) = 1/x ↑ بشرط x > 0

⚙️ قواعد الاشتقاق المركّب

قاعدة الضرب

(u·v)' = u'·v + u·v' — اشتقاق الأول × الثاني + الأول × اشتقاق الثاني

قاعدة القسمة

(u/v)' = (u'v - uv') / v² — سهّلها: "فوق'×تحت - فوق×تحت' على تحت²"

قاعدة السلسلة

(f(g(x)))' = f'(g(x)) · g'(x) — اشتقاق الخارجية مضروباً في اشتقاق الداخلية

✏️ مثال — قاعدة السلسلة

جد مشتقة: f(x) = sin(x²)

  1. الدالة الخارجية: sin، الداخلية: x²
  2. نشتق الخارجية: cos(x²)
  3. نضربها في مشتقة الداخلية: 2x
  4. النتيجة: f'(x) = 2x·cos(x²)
💡

حيلة ذهنية: قاعدة السلسلة = "اشتق من الخارج للداخل، وكل مرة تشتق دالة داخلية اضرب بمشتقتها"

📈 تطبيقات الاشتقاق

إيجاد الميل عند نقطة معادلة المماس القيم العظمى والصغرى معدل التغير رسم الدوال

∫ التكامل وتطبيقاته

التكامل هو عملية عكس الاشتقاق. يُستخدم لإيجاد المساحات والحجوم وكثير من التطبيقات العملية.

النوع الأول: التكامل غير المحدود

التكامل غير المحدود
∫ f(x) dx = F(x) + C
C: ثابت التكامل — لا تنساه أبداً!

📋 جدول التكاملات الأساسية

الثابت ∫ c dx = cx + C ∫ 5dx = 5x + C
الأس ∫ xⁿ dx = xⁿ⁺¹/(n+1) + C n ≠ -1
1/x ∫ (1/x) dx = ln|x| + C ↑ الحالة n = -1
eˣ ∫ eˣ dx = eˣ + C ↑ تتكامل لنفسها
sinx ∫ sinx dx = -cosx + C ↑ لاحظ السالب
cosx ∫ cosx dx = sinx + C ↑ بدون سالب

النوع الثاني: التكامل المحدود

مبرهنة الحساب الأساسية
∫[a,b] f(x)dx = F(b) - F(a)
نعوّض الحد الأعلى ثم نطرح منه قيمة الحد الأدنى
✏️ مثال محلول — تكامل محدود

احسب: ∫[0,2] (3x² + 2x) dx

  1. نجد التكامل: F(x) = x³ + x²
  2. نطبق المبرهنة: F(2) - F(0)
  3. F(2) = 8 + 4 = 12
  4. F(0) = 0 + 0 = 0
  5. النتيجة = 12
⚠️

في التكامل غير المحدود لا تنسَ ثابت التكامل +C — إهماله خطأ شائع يُخصم عليه في الامتحانات!

🛠️ تقنيات التكامل

التكامل بالتعويض

إذا كان لديك ∫f(g(x))·g'(x)dx، ضع u = g(x) وعوّض. مفيد جداً عندما تجد الدالة ومشتقتها معاً.

التكامل بالأجزاء

∫u·dv = u·v - ∫v·du — استخدمه عندما يكون لديك حاصل ضرب دالتين مختلفتين مثل x·sinx

حساب المساحة بين منحنيين حجم الجسم الدوراني التكامل بالكسور الجزئية
